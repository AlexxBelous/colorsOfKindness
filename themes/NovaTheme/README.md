# NovaTheme 🚀

## 📌 Версионность и Кэширование
Текущая версия темы: **1.0.0**

При внесении крупных изменений в стили или скрипты (особенно для Production):
1. Обновите версию в `style.css` (в шапке темы).
2. Обновите версию в `functions.php` (переменная `$theme_version`), чтобы сбросить кэш у пользователей.
---

Современная стартовая тема для WordPress с использованием **Vite**, **Sass** и **React**.

## 🛠 Стек технологий
* **PHP / WordPress** — серверная часть.
* **Vite 5+** — сборщик фронтенда (мгновенный HMR).
* **Sass (SCSS)** — препроцессор для стилей.
* **React 18+** — для создания интерактивных интерфейсов (опционально).
* **Ubuntu Linux** — рекомендуемая среда разработки.

---

## 🚀 Быстрый старт (Разработка)

1.  **Установка зависимостей:**
    В терминале из папки темы выполните:
    ```bash
    npm install
    ```

2.  **Запуск сервера разработки:**
    ```bash
    npm run dev
    ```
    *Vite запустится на `http://localhost:3000`. Тема автоматически переключится в режим разработки через константу `IS_VITE_DEVELOPMENT` в `functions.php`.*

3.  **Локальный домен:**
    Убедитесь, что ваш локальный сервер WordPress (Apache/Nginx) запущен. Тема будет подтягивать скрипты с порта 3000 и внедрять их в ваш PHP-шаблон.

---

## 📦 Сборка для Production

Когда тема готова к деплою на реальный сервер:

1.  **Запустите билд:**
    ```bash
    npm run build
    ```
    *Vite создаст оптимизированные файлы в папке `/assets`.*

2.  **Переключите режим в `functions.php`:**
    Установите значение константы в `false`:
    ```php
    define('IS_VITE_DEVELOPMENT', false);
    ```

## ⚛️ Работа с React

В текущем состоянии **React временно отключен**, чтобы не нагружать проект на этапе базовой верстки.

### Как включить React (Активация):

Если вам понадобилось создать React-компонент, выполните 3 шага:

1.  **В `src/js/main.jsx`:**
    * Раскомментируйте строку: `import App from './App.jsx';`
    * Раскомментируйте блок `if (document.getElementById('root')) { ... }`.

2.  **В `functions.php`:**
    * Найдите и раскомментируйте хук:
        ```php
        add_action('wp_head', 'novatheme_vite_head_preamble');
        ```
      *Это включит React Refresh Runtime, необходимый для работы Vite.*

3.  **В HTML-шаблоне (например, `index.php`):**
    * Убедитесь, что на странице есть контейнер: `<div id="root"></div>`.

## 📂 Структура файлов
* `src/scss/` — исходники стилей. Главный файл: `main.scss`.
* `src/js/` — исходники JavaScript. Главный файл: `main.jsx`.
* `src/js/global.js` — место для обычного Vanilla JS (меню, скроллы и т.д.).
* `assets/` — скомпилированные файлы (создаются автоматически после `npm run build`).

---

## 📝 Заметки для разработчика (Ubuntu)
* При добавлении новых PHP-файлов Vite автоматически перезагрузит страницу благодаря плагину `vite-plugin-live-reload`.
* Если возникают проблемы с правами доступа к папке `assets`, проверьте владельца директории через `chown`.
